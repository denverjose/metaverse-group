# Static One-Page Website

This project is a modern static one-page website built with **Next.js**, **Tailwind CSS**, and **Framer Motion**.

It is designed to be clean, fast, and fully responsive across desktop, tablet, and mobile devices.

## Tech Stack

- **Next.js** – React framework for optimized performance
- **Tailwind CSS** – Utility-first styling
- **Framer Motion** – Smooth animations and transitions

## Features

- Single-page layout (no external routes)
- Fully responsive design
- Smooth scroll and animation effects
- Clean and modern UI

## Getting Started

### Environment Variables

This project requires a `.env` file to run properly. The `.env` file contains configuration values and environment-specific settings.

**Important:** The `.env` file is not included in this repository for security reasons.

If you would like to run the project locally, please **contact the project owner (me)** to request the required `.env` file.

Once you receive the `.env` file:

1. Place it in the root directory of the project.
2. Install dependencies:
   ```bash
   npm install
    ```
3. Start the development server:

    ```bash
    npm run dev
    ```

4. Then open:

    ```
    http://localhost:3000
    ```
